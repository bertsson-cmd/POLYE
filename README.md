# PolyEdge 95 v2

Polymarket opportunity scanner + paper-trading engine. Four strategies
(Dutch-book arbitrage, correlated-market locks, longshot fading, convergence
yield), fractional-Kelly risk management, and a Windows 95 terminal dashboard
published to GitHub Pages every 30 minutes.

**Read `MANUAL.md` — full setup and operating instructions.**

Quick start: enable Actions write permissions + GitHub Pages (`/docs` folder),
delete `state/paper_state.json` to clear the demo data, run the workflow.
